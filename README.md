# Branding

## Overview
Official brand assets, guidelines and design resources for Limelight IT Group.

## Purpose
Single source of truth for all Limelight IT brand materials. Covers logos, colour palettes, typography, templates and usage guidelines. All client-facing materials should reference this repo for brand consistency.

## Structure
```
branding/
├── logos/         # Official logo files (PNG, SVG, various sizes)
├── colours/       # Colour palette documentation and swatches
├── typography/    # Font files and typographic guidelines
├── templates/     # Branded templates (presentations, documents, social)
├── guidelines/    # Brand usage guidelines and standards
├── .gitignore
└── README.md
```

## Brand Colours
| Name          | Hex       | Usage                        |
|---------------|-----------|------------------------------|
| Lime Glow     | `#C6FF3E` | Primary accent, CTA elements |
| Charcoal      | `#0E0E0E` | Primary background           |
| Deep Navy     | `#0A1A2F` | Secondary background         |
| Graphite Grey | `#1F1F1F` | Card backgrounds             |
| Soft Grey     | `#D9D9D9` | Body text on dark            |

## Typography
- **Headings:** Montserrat (700/800)
- **Body:** Inter (400/500)
- **Mono/Tags:** Roboto Mono (400/600)

## Standards
- Always use official logo files — do not recreate from scratch
- Maintain minimum clear space around the logo equal to the height of the "L" in LIMELIGHT
- Never stretch or recolour the logo outside brand guidelines
- All new assets must be added in both PNG and SVG where possible
